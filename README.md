# Food Delivery Analytics System

A comprehensive Java-based Food Delivery Analytics System designed to simulate real-world food delivery operations while demonstrating advanced Java programming concepts and modern software development practices.

This project provides end-to-end functionality for validating orders, analyzing customer and restaurant performance, processing data asynchronously, generating reports, implementing dynamic search capabilities, performing date-time analytics, and managing in-memory caching. The application leverages Java Stream API, Functional Interfaces, Generics, CompletableFuture, Concurrent Collections, and JUnit Testing to build a scalable and modular analytics platform.

* Order amount validation
